# csv-transform
A scriptable tool to transform data in csv files.

### Why use this script?
This script is designed to be scriptable with ease of use in mind. You should
use this script if you need to automate the transformation of csv data in some
data processing pipeline.

### Usage
See example for how to use.

### Running example
Change into src/ directory and execute:
```
python3 example.py example_input.csv output.csv
```
Output should be written into src/output.csv. Inspect src/example.py to see
how output was created. Copy and adjust src/example.py for your own use.
